사용방법

※ 이미지를 클릭하면 영상으로 연결됩니다.

    Adominal

  1. 의자에 똑바로 앉아 가슴패드가 가슴 앞에 오도록 패드를 조절합니다
  2. 짧은 동작으로 가슴 패드를 아래로 밀어 구부립니다
  3. 발과 다리를 넓게 벌릴수록 복근을 더욱 강화할 수 있습니다
  4. 처음 위치로 되돌아 온 후 같은 동작을 반복합니다
    5. target muscle : 복직근

    Rotary Torso

  1. 의자에 앉아 양쪽 팔걸이를 오른쪽 또는 왼쪽으로 돌리십시오. 의자에 앉을 떄와 운동을 마친 후 의자로부터 일어날 때에만 팔걸이를 중앙에 위치하도록 합니다
  2. 2개의 롤러패드 사이에 각각 팔을 끼우고 2개의 손잡이 중 편한 쪽을 잡으십시오
  3. 가능한한 멀리 몸을 반대방향으로 돌리십시오. 처음위치로 되돌아온 후 같은 동작을 반복합니다
  4. 한쪽 편 운동을 마친 후, 팔걸이를 반대방향으로 돌리고 반대쪽 근육 운동을 한 후 팔걸이를 중앙에 위치시키고 의자에서 일어나십시오
    5. target muscle : 외복사근

    Lat Pulldown

  1. 롤러패드가 허벅지 위쪽에 놓이도록 높이를 조절하고 레트바 양쪽 그립을 잡고 롤러패드에 허벅지를 조심스럽게 끼우면서 앉습니다
  2. 래트바를 가슴 앞쪽을 향해서 아래로 당깁니다
  3. 이떄, 승모근이 올라가지 않도록 주의합니다
  4. 처음 위치로 되돌아 온 후 같은 동작을 반복합니다
    5. target muscle : 광배근

    Seated Low

  1. 패드가 가슴 중앙에 오도록 의자 높이를 조절한 후, 팔을 뻗어 핸들이 닿을 수 있도록 가슴 패드를 조절하십시오
  2. 발을 발받침대에 놓고 앉아 두가지 중 한 핸들을 잡으시오
  3. 전완이 바닥과 평행을 유지하도록 하면서 가슴쪽으로 당깁니다. 이때 승모근이 긴장되지 않도록 주의하며 팔꿈치가 밖으로 빠지지 않도록 합니다
  4. 처음 위치로 되돌아온 후 같은 동작을 반복합니다
    5. target muscle : 광배근

    Shoulder Press

  1. 두 종류의 핸들 중 편한 쪽을 잡으시오
  2. 이때, 팔꿈치는 가슴과 수평 또는 약간 앞을 향하도록하고, 상완은 바닥과 평행이 되도록 합니다
  3. 팔을 위로 향하여 뻗어주십시오
  4. 처음위치로 되돌아 온 후 같은 동작을 반복합니다
    5. target muscle : 삼각근

    Reverse Pec-deck Fly

  1. 위의 핀을 당겨 팔걸이를 P 또는 1, 2, 3의 위치에 고정합니다
  2. 팔이 바닥과 평행이 되도록 의자 높이를 조절합니다
  3. 위 또는 아래 손잡이를 잡고 양팔이 가능한 뒤로 당기십시오
  4. 처음 위치로 되돌아 온 후 같은 동작을 반복합니다
    5. target muscle : 후면 삼각근

    Seated Chest

  1. 페달을 밟아 좌우 핸들이 앞으로 오면, 2종류의 핸들 중 편한 쪽을 택하여 잡은 후 페달에서 발을 떼십시오
  2. 이때 팔은 가슴높이에서 바닥과 평행이 되도록 합니다
  3. 팔꿈치의 각도가 90도가 되도록 팔꿈치를 굽힙니다
  4. 처음 위치로 되돌아 온 후 같은 동작을 반복합니다
    5. target muscle : 대흉근

    Pec-deck Fly

  1. 위의 핀을 당겨 팔걸이를 7, 8 또는 6에 맞추고 팔이 바닥과 평행이 되도록 의자높이를 조절합니다
  2. 아래쪽 손잡이를 잡고 두 손이 닿을 때까지 가운데 앞으로 당기십시오
  3. 처음 위치로 되돌아 온 후 같은 동작을 반복합니다
    4. target muscle : 대흉근

    Hack Squaw

  1. 어깨 패드에 어깨를 가져다 댑니다
  2. 발판에 양발을 어깨너비로 잘 정렬합니다
  3. 엉덩이가 정강이 높이까지 많이 내려가지 않게 주의하며 앉습니다
  4. 처음 위치로 되돌아 온 후 같은 동작을 반복합니다
    5. target muscle : 대퇴근, 둔근

    Leg Press

  1. 등받이를 뒤로 밀고 의자에 앉아 발받침에 발을 올려 놓습니다
  2. 이때, 양발의 간격은 어깨넓이만큼 벌리고 발뒤꿈치가 발받침 하단으로부터 3~5cm 위에 놓이도록 합니다. 발을 더 위에 놓으면 대퇴사두근을 더욱 강화시킬 수 있습니다
  3. 레버를 당겨 무릎이 90도가 되도록 의자 위치를 조절한 후 앞뒤로 움직여 핀이 가까운 구멍에 걸리도록 한 후 핸들을 잡습니다
  4. 다리와 엉덩이로 발받침을 미십시오. 무릎은 완전히 펴지않고 마지막 동작에서 약간 굽어야합니다
  5. 처음 위치로 되돌아 온 후 같은 동작을 반복합니다
    6. target muscle : 대퇴근, 둔근

    Leg Extension

  1. 왼편에 있는 신장조견표에 따라 등받이를 조절합니다
  2. 기계의 회전축과 무릎축이 일치하도록 앉았을 떄 정강이 패드가 발목 앞에 오도록 패드 길이를 조절합니다
  3. 양쪽에 있는 핸들을 잡고 무릎이 무리 하지 않는 선까지 다리로 정강이 패드를 들어올립니다
  4. 처음 위치로 되돌아 온 후 같은 동작을 반복합니다
    5. target muscle : 대퇴근

    Leg Curl

  1. 기계의 회전축과 무릎축이 일치하도록 엎드렸을 떄 패드가 발목 뒤에 오도록 패드 길이를 조절합니다
  2. 암패드에 팔을 올리고 손잡이를 잡습니다
  3. 다리로 패드를 들어 엉덩이 뒤로 당깁니다
  4. 처음 위치로 되돌아 온 후 같은 동작을 반복합니다
    5. target muscle : 넙다리두갈래근

    Multi Hip

  1. 발판에 서서 앞쪽에 있는 손잡이를 잡고, 패드위에 운동할 허벅지 뒷부분을 살짝 올려놓습니다
  2. 발끝을 하늘로 올리는 느낌으로 골반이 움직이지 않을 정도까지 올립니다
  3. 이때, 골반이 상하좌우로 움직이지 않도록 주의합니다
  4. 처음위치로 되돌아 온 후 같은 동작을 반복합니다
    5. target muscle : 둔근

    Biceps Curl

  1. 발을 앞으로 놓고 앉으면서 핸들을 잡습니다
  2. 의자 높이 위 또는 아래로 1단씩 조절하면서 운동범위를 조절할 수 있습니다 또한 손잡이를 잡는 손의 각도와 간격을 조절하면 강화되는 근육의 범위를 변경할 수 있습니다
  3. 팔을 위로 구부립니다
  4. 처음 위치로 되돌아 온 후 같은 동작을 반복합니다
    5. target muscle : 이두근

    Overhead Tricep Extenstion

  1. 의자에 앉아 발을 발 받침에 놓은 후 보조 페달을 밟으십시오
  2. 양손을 머리 뒤로 올려 핸들을 잡은 후 보조 페달에서 발을 떼십시오
  3. 등을 등받이에 고정하고 팔꿈치는 앞을 향한 자세에서 팔을 위로 올리십시오
  4. 처음 위치로 되돌아 온 후 같은 동작을 반복합니다
    5. target muscle : 삼두근
